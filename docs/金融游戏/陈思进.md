## 欠多少，赚多少，花多少

### 储蓄

- 花的比赚的少
- 结余的钱要投资，不闲置
- 避免债务，尤其高息债务

### 负债

|       显性负债       |           隐形负债            |
|:----------------:|:-------------------------:|
| 房贷 (要小于税后收入的30%) |    信用卡 （循环还款，每次最小金额还款）    |
|        车贷        |           亲友借款            |
|     消费贷、经营贷      | 未结清应付款 （分期购物，水电燃气，物业，车保养） |
|  其他贷款 （装修贷，助学贷）  |   或有负债 （担保贷款，未决诉讼，未交税费）   |

### 家庭健康红线

- 整体负债率 = 总负债率 / 总资产 x 100%  (是否小于40%)
- 月供压力 = 月负债还款额 / 家庭月税后总收入 x 100% (是否小于 30%)
- 月净现金流 = 月均可持续收入 - 月均总支出  (正可聚财，负则危机)
  - 月净现金流为正 -> 连续三月为正 -> 净现金流结余 > 月均可持续收入20%
- 无高息负债, 年化利率 > 8% 为高息
- 固定支出小于月收入50%，如还贷，衣食住行
- 储蓄20%用于定投，强制储蓄，减少冲动消费
- 储备3~6~12月开支应急

#### 家庭主动和被动收入

|   主动收入   |     被动收入     |
|:--------:|:------------:|
|    工资    |     理财收入     |
| 奖金 （年终奖） |     房租收入     |
| 劳务报酬（兼职） |   知识产权（版权）   |
|  经营性收入   | 被动经营  （加盟投资） |

#### 可持续收入三大维度

- 稳定性：持续收入
- 可复制性抗风险：多份工资， 趋势学习
  - AI 应用落地
  - 康养护理，慢病管理，健康监测等养老产业
  - 双碳下节能环保
  - 数字化服务，短视频，财税等
  - 区域化跨境电商，中东
- 合理性：被动收入占总资产20%，并提升

#### 支出

| 必要支出 |    非必要支出     |
|:----:|:------------:|
|  伙食  |     娱乐休闲     |
|  住房  |     冲动消费     |
|  交通  |    闲置性支出     |
| 医疗支出 |    攀比性支出     |
| 债务月供 | 低效支出 （买课不去上） |
| 子女教育 |              |

#### 记账要点
- 全面记录：线上线下，不论金额大小
- 分类记录：按必要和非必要
- 定期统计：每周统计、每月总结，削减非必要
- 坚持到底

## 隐性财务信息
- 保障类 => 五险一金 => 养老保障
- 收入潜力类 => 行业周期性，晋升+积累+涨薪幅度，持续迭代技能，兼职副业  => 家庭收入弹性
- 资产类 => 房产(物业，维修，房产税，空置租金损失)、车辆(保险，保养，停车费，违章罚款，折旧，油费)的隐性成本，闲置的机会成本 (活期利率低)  => 财务积累的效率
- 法律税务类 => 个税抵扣， 产权归属清晰， 担保债务风险， 税务申报合规 => 财富合规合法性

## 存多少心里不慌
不要把应急的钱拿去投资，一定要专款专用
- 单身/丁克 （收入稳定）： 3~6个月固定支出
- 有孩子/家庭负债高(负债率>30%)： 6~12个月固定支出
- 收入不稳定（自由职业，个体户，高危职业）：12~18个月固定支出

### 提高应急资金的收益
- 即时可用 占20% ~ 30%： 货币基金，应对一周内的突发需求 （年化1.5%~2%）
- 快速可用 占50% ~ 60%：银行智能存款，应对1周~3个月内的需求，核心保证 （年化1.8%~2.5%）
- 收益增强 占10% ~ 20%：短期国债逆回购/结构性存款，应对三个月内不用的部分 （年化2% ~ 3%）

### 医疗保证
- 刚需保障：医保+百万医疗险（报销医保后的大额支出）+重疾（保证病后生活，年收入3~5倍保额）
- 小额保障：门诊险+意外险
> 保险的最高境界是大家永远用不到

### 房屋车辆维修
- 建立专项储蓄账户，即单独的银行卡或者理财账户
- 确定每月存款额度
  - 房屋：房屋总价 * 0.1% 或者 0.2% 每月
    - 新房0.1，老房（5年以上）0.2，有维修基金的，可减少额度。
  - 车辆：车辆总价 * 0.1% 或者 0.2% 每月
    - 新车0.1，老车（5年以上）0.2，有保险的可适当减少额度

### 其他大额支出
- 意外赔偿：意外险，车主责任险，家庭财产险等转移
- 失业后的长期开支：应急资金+低息理财

## 教育、养老、退休要准备多少钱
### 教育
教育通胀3%~5%每年
- 国内从小到大学毕业开支，涵盖学费，课本费，兴趣班，补课费，生活费等
  - 学前：3~6岁
    - 公办：3~5w
    - 民办：10~30w
  - 义务教育 6~15岁
    - 课本费 5k~1w + 补课10w/year + 学区溢价
  - 高中15~18岁
    - 公办 1~2w + 补课费10w/year
    - 国际高中：30w ~ 60w
  - 大学和研究生18~24岁
    - 公办四年 16w ~ 32w

### 养老
- 预期寿命：85~90、 95~100
- 退休生活标准：退休后的收入占目前刚性需求的90%+
- 医疗开支：每月医疗支出 x (1+0.04)^(年 距离多久退休)
- 通胀率: 每月所需生活费 x (1+0.035)^(年 距离多久退休)
- 现有养老金打底：基础养老金+个人养老金
